랭킹 학습

term_id: learning_to_rank

Category: training_techniques

Definition

표준 분류나 회귀와 달리 랭킹 학습은 항목들의 상대적 순서를 예측하는 데 중점을 둡니다. N… 같은 랭킹 오류를 최소화하기 위해 쌍별, 목록별, 또는 개별 접근 방식을 사용합니다.

Summary

랭킹 학습은 주어진 쿼리에 대한 관련성에 따라 항목을 순서대로 정렬하는 데 사용되는 지도 기계 학습 기법으로, 검색 엔진에서 흔히 사용됩니다.

Key Concepts

  • NDCG (평균 정규화 누적 이득)
  • 쌍별 랭킹
  • 정보 검색

Use Cases

  • 검색 엔진 결과 순서 매기기
  • 제품 추천 시스템
  • 광고 랭킹 알고리즘

Related Terms

  • Information retrieval (정보 검색)
  • NDCG (평균 정규화 누적 이득)
  • Recommendation systems (추천 시스템)

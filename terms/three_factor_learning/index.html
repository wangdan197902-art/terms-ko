Three-factor learning

term_id: three_factor_learning

Category: training_techniques

Definition

3요인 학습(Three-factor learning)은 강화학습 내에서 학습 과정을 세 가지 명확한 구성 요소로 분해하는 특정 접근 방식입니다. 즉, 보상 신호(Reward Signal), 가치 함수(Value Function), 정책(Policy)으로 구성되며…

Summary

보상, 가치, 정책 요소를 활용하여 에이전트의 행동을 최적화하는 강화학습 프레임워크입니다.

Key Concepts

  • 보상 신호
  • 가치 함수
  • 정책 최적화
  • 강화학습

Use Cases

  • 게임 에이전트
  • 로봇 제어
  • 자원 관리

Related Terms

  • Reinforcement Learning (강화학습: 에이전트가 환경과 상호작용하며 보상을 최대화하도록 학습하는 방법)
  • Q-Learning (Q러닝: 상태와 행동 쌍의 가치를 학습하는 비모델 기반 강화학습 알고리즘)
  • Policy Gradient (정책 경사: 직접 정책을 매개변수화하여 최적화하는 강화학습 방법)
  • Reward Shaping (보상 설계: 에이전트의 학습 속도를 높이기 위해 보상 함수를 수정하는 기법)

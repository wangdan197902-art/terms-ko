검색(Retrieval)

term_id: retrieval

Category: application_paradigms

Definition

검색은 사용자 쿼리나 맥락을 바탕으로 대규모 데이터셋이나 외부 지식 기반에서 특정 정보를 검색하고 추출하는 기술적 과정을 의미합니다. 현대 AI 시스템에서는 특히 RAG(Retrieval-Augmented Generation)와 같은 방식에서 모델의 정확성과 사실 기반 응답 능력을 높이는 데 핵심적으로 활용됩니다.

Summary

모델의 입력을 보강하기 위해 데이터베이스나 지식 기반에서 관련 데이터를 가져오는 과정입니다.

Key Concepts

  • 정보 검색
  • 벡터 검색
  • 색인(Indexing)
  • 랭킹(Ranking)

Use Cases

  • 고객 지원 챗봇
  • 문서 검색 엔진
  • 추천 시스템

Code Example

1

null

Related Terms

  • rag (RAG: 검색 증강 생성, 외부 정보를 활용해 생성 모델의 성능 향상)
  • vector_database (벡터 데이터베이스: 임베딩된 데이터를 효율적으로 저장하고 검색하는 DB)
  • semantic_search (시맨틱 검색: 키워드가 아닌 의미적 유사성을 기반으로 검색)
  • embedding (임베딩: 텍스트나 이미지를 고차원 벡터로 변환하는 기술)

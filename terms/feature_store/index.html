피처 스토어

term_id: feature_store

Category: basic_concepts

Definition

피처 스토어는 데이터 엔지니어링 팀과 머신러닝 팀 간의 가교 역할을 하며, 배치 훈련과 실시간 추론 모두를 위한 피처의 통합된 뷰를 제공합니다. 이를 통해 일관성을 보장합니다.

Summary

머신러닝 훈련과 추론 전반에 걸쳐 피처를 일관되게 관리, 공유 및 제공하기 위해 설계된 중앙 집중식 저장소입니다.

Key Concepts

  • 훈련-추론 편차(Training-serving skew)
  • 중앙 집중식 관리
  • 피처 버전 관리
  • MLOps 인프라

Use Cases

  • 엔터프라이즈 ML 플랫폼
  • 실시간 사기 탐지 시스템
  • 협업 모델 개발 팀

Related Terms

  • MLOps (머신러닝 운영)
  • Data Pipeline (데이터 파이프라인)
  • Model Serving (모델 서빙)
  • Feature Engineering (피처 공학)

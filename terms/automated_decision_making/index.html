자동화된 의사결정

term_id: automated_decision_making

Category: basic_concepts

Definition

자동화된 의사결정(ADM)은 이전에 인간의 판단이 필요했던 선택을 소프트웨어 시스템을 통해 수행합니다. 신용 점수 산정, 콘텐츠 검토, 물류 등에서 흔히 사용되며, ADM은 미리 정의된 규칙을 사용합니다.

Summary

알고리즘이 직접적인 인간의 감독 없이 데이터 입력에 기반하여 독립적으로 행동이나 결과를 선택하는 과정입니다.

Key Concepts

  • 알고리즘 거버넌스
  • 효율성
  • 편향 완화
  • 인간 개입 루프(Human-in-the-Loop)

Use Cases

  • 은행의 신용 승인 시스템
  • 전자상거래의 동적 가격 책정
  • 의료 분야의 분류 시스템(Triage systems)

Related Terms

  • Algorithmic Bias (알고리즘 편향)
  • Decision Support Systems (의사결정 지원 시스템)
  • Rule-Based Systems (규칙 기반 시스템)
  • Governance (거버넌스)

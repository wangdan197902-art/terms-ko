피드백

term_id: feedback

Category: basic_concepts

Definition

피드백 메커니즘은 AI 시스템이 사용자나 환경과의 상호작용으로부터 학습하여 향후 예측이나 행동을 정교하게 다듬도록 합니다. 여기에는 강화 학습 신호, 인간 개입(human-in-the-loo…) 등이 포함됩니다.

Summary

피드백은 출력 결과를 사용하여 AI 모델이나 시스템의 성능을 반복적으로 조정하고 개선하는 과정을 포함합니다.

Key Concepts

  • 강화 학습
  • 인간 개입
  • 오류 수정
  • 반복적 개선

Use Cases

  • 추천 시스템 튜닝
  • 챗봇 대화 정제
  • 로봇 제어 조정

Related Terms

  • Reinforcement Learning (강화 학습)
  • Supervised Learning (지도 학습)
  • Reward Function (보상 함수)

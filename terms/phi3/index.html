Phi3

term_id: phi3

Category: basic_concepts

Definition

Phi-3은 마이크로소프트가 출시한 소형 언어 모델(SLM) 시리즈로, 훨씬 더 큰 모델들과 비교해도 견줄 만한 높은 성능을 제공하면서도 필요한 계산 자원은 현저히 적게 요구하도록 설계되었습니다. 이는 고품질의 합성 데이터를 활용한 훈련을 통해 달성됩니다.

Summary

마이크로소프트 리서치에서 개발한 오픈 웨이트(Open-weight) 대규모 언어 모델 계열.

Key Concepts

  • 소형 언어 모델(Small Language Models)
  • 마이크로소프트 리서치(Microsoft Research)
  • 엣지 AI(Edge AI)
  • 합성 데이터 훈련(Synthetic Data Training)

Use Cases

  • 모바일 기기 또는 노트북에서 LLM 실행
  • 기업 애플리케이션을 위한 비용 효율적인 추론(Inference)
  • 낮은 지연 시간(Latency)이 요구되는 실시간 처리

Related Terms

  • LLM (대규모 언어 모델)
  • On-device AI (온디바이스 AI)
  • Transformer architecture (트랜스포머 아키텍처)
  • Distillation (디스틸레이션/지식 증류)

TFLite

term_id: tflite

Category: basic_concepts

Definition

TensorFlow Lite는 스마트폰, 마이크로컨트롤러, IoT 장치 등 자원이 제한된 디바이스에 머신러닝 모델을 배포하기 위해 설계된 오픈소스 프레임워크입니다. 이 프레임워크는 모델 최적화를 통해…

Summary

TensorFlow Lite(TFLite)는 머신러닝 모델을 모바일, 임베디드 및 엣지 디바이스에서 실행할 수 있게 해주는 도구 모음입니다.

Key Concepts

  • 모델 양자화
  • 엣지 컴퓨팅
  • 추론 최적화
  • 모바일 배포

Use Cases

  • 온디바이스 이미지 인식
  • AR 앱에서의 실시간 객체 감지
  • 스마트 스피커의 음성 명령 처리

Related Terms

  • tensorflow (텐서플로우: 구글에서 개발한 오픈소스 머신러닝 라이브러리)
  • edge_ai (엣지 AI: 중앙 서버가 아닌 단말기에서 AI를 실행하는 기술)
  • model_quantization (모델 양자화: 모델의 정밀도를 낮춰 크기와 연산 속도를 개선하는 기법)

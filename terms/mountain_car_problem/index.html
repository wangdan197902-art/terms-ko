마운틴 카 문제

term_id: mountain_car_problem

Category: basic_concepts

Definition

마운틴 카 문제는 강화 학습 연구의 표준 벤치마크입니다. 목표는 출력이 부족한 자동차를 제어하여 가파른 언덕 꼭대기에 도달하는 것입니다. 자동차가 한 번에 언덕을 오를 수 없기 때문에, 에이전트는 중력을 이기고 속도를 얻기 위해 왕복 운동을 해야 합니다.

Summary

에이전트가 가속 제어 장치만 사용하여 가파른 언덕을 올라가야 하는 고전적인 강화 학습 과제입니다.

Key Concepts

  • 희소 보상
  • 지연된 결과
  • 연속 제어
  • 벤치마킹

Use Cases

  • 새로운 강화 학습 알고리즘 테스트
  • 값 함수 근사 기법 시연
  • 강화 학습 개념 교육용 예제

Related Terms

  • OpenAI Gym (강화 학습 환경 라이브러리)
  • 강화 학습 (보상을 통해 학습하는 AI)
  • 진자 스윙업 (매달린 물체를 세워 균형 잡기 문제)
  • 값 반복 (동적 프로그래밍 알고리즘)

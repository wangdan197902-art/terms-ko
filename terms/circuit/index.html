회로

term_id: circuit

Category: basic_concepts

Definition

AI 맥락에서 회로는 일반적으로 행렬 연산과 병렬 처리를 가속화하도록 설계된 GPU, TPU 또는 뉴로모픽 칩과 같은 하위 하드웨어 아키텍처를 나타냅니다. 이러한

Summary

회로는 연산 작업을 실행하는 칩 및 인터커넥트를 포함한 물리적 하드웨어 인프라를 참조합니다.

Key Concepts

  • 하드웨어 가속
  • 병렬 처리
  • 뉴로모픽 컴퓨팅
  • GPU/TPU 아키텍처

Use Cases

  • 대규모 신경망 학습
  • 실시간 추론 배포
  • 에너지 효율적인 엣지 컴퓨팅

Related Terms

  • GPU (그래픽 처리 장치)
  • TPU (텐서 처리 장치)
  • Neural Network (신경망)
  • Compute Cluster (컴퓨팅 클러스터)

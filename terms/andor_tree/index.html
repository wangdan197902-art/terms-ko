AND-OR 트리(And–or tree)

term_id: andor_tree

Category: basic_concepts

Definition

AND-OR 트리는 문제 해결 및 계획 수립, 특히 AI 검색 알고리즘에서 사용되는 표현 방식입니다. ‘OR’ 노드는 서로 다른 행동 간의 선택을 나타내며, ‘AND’ 노드는 한

Summary

검색 알고리즘에서 사용되는 계층적 그래프 구조로, 노드는 상태를 나타내고 간선은 부분 목표에 도달하기 위한 행동을 나타냅니다.

Key Concepts

  • 검색 알고리즘
  • 문제 분해(Problem Decomposition)
  • AO* 알고리즘
  • 상태 공간(State Space)

Use Cases

  • 자동화된 계획 수립(Automated Planning)
  • 게임 트리 탐색(Game Tree Search)
  • 복잡한 문제 해결

Related Terms

  • 마르시맥스 알고리즘 (Minimax algorithm)
  • 상태 공간 탐색 (State space search)
  • 계획 그래프 (Planning graphs)
  • 의사결정나무 (Decision trees)

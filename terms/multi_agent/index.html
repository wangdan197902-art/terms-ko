다중 에이전트

term_id: multi_agent

Category: application_paradigms

Definition

다중 에이전트 시스템은 환경을 지각하고, 결정을 내리며, 행동에 옮기는 여러 독립적이고 지능적인 엔티티로 구성됩니다. 이러한 에이전트들은 서로 협력하거나 경쟁하거나 협상할 수 있습니다.

Summary

여러 자율 에이전트가 환경 내에서 상호작용하여 개별적 또는 집단적 목표를 달성하는 시스템 아키텍처입니다.

Key Concepts

  • 자율성
  • 상호작용
  • 조정
  • 분산 지능

Use Cases

  • 자율 주행 차량 교통 조정
  • 공급망 최적화 시뮬레이션
  • 멀티플레이어 게임 AI 개발

Related Terms

  • swarm intelligence (군집 지능)
  • cooperative AI (협력형 AI)
  • game theory (게임 이론)
  • distributed systems (분산 시스템)

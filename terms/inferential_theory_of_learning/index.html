추론적 학습 이론

term_id: inferential_theory_of_learning

Category: training_techniques

Definition

이 이론은 학습이 본질적으로 확률적 추론 과정이라고 주장합니다. 학습자는 데이터를 암기하는 대신 가능한 모델이나 가설들에 대한 확률 분포를 유지합니다. 새로운 데이터가 관측되면 베이즈 정리를 사용하여 이 분포를 업데이트합니다.

Summary

학습을 베이지안 추론으로 보고, 관측된 데이터를 바탕으로 가설에 대한 신념을 업데이트하는 프레임워크.

Key Concepts

  • 베이지안 추론
  • 사전 분포
  • 사후 업데이트
  • 모델 불확실성

Use Cases

  • 제한된 데이터 하의 강건한 예측
  • 능동 학습 전략
  • 불확실성 인식 의사결정

Related Terms

  • Bayesian optimization (베이지안 최적화)
  • Variational inference (변분 추론)
  • Probabilistic graphical models (확률적 그래픽 모델)

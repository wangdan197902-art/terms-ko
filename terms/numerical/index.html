수치적

term_id: numerical

Category: basic_concepts

Definition

AI 및 데이터 과학의 맥락에서 수치적(numerical)은 정수, 부동 소수점, 소수 등 정량적 값을 포함하는 데이터 유형이나 방법을 지칭합니다. 범주형이나 텍스트 데이터와 달리 수치적 데이터는…

Summary

기호 논리나 텍스트가 아닌 숫자 또는 수학적 계산을 관련하거나 포함하거나 사용하는 것.

Key Concepts

  • 부동 소수점
  • 정수
  • 데이터 유형
  • 정량적 데이터

Use Cases

  • 통계 모델링
  • 금융 예측
  • 센서 데이터 처리

Code Example

1
2
3
4

import numpy as np
arr = np.array([1.5, 2.7, 3.9])
mean_val = np.mean(arr)
print(mean_val)

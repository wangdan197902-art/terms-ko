피처 (Feature)

term_id: feature

Category: basic_concepts

Definition

머신러닝에서 피처는 데이터셋 내의 특정 사례(instance)를 설명하는 고유한 속성이나 변수입니다. 피처는 수치형, 범주형, 텍스트형 등 다양한 형태를 가질 수 있으며, 머신러닝 모델의 가장 기본적인 입력 요소 역할을 합니다.

Summary

관측되는 현상의 개별 측정 가능한 속성 또는 특성으로, 머신러닝 모델에 입력 데이터로 사용됩니다.

Key Concepts

  • 입력 변수
  • 속성
  • 데이터셋 열
  • ML 입력

Use Cases

  • 예측 모델링의 입력값
  • 데이터 분석 및 시각화
  • 피처 선택 과정

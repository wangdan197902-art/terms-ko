파운데이션 모델(Foundation Model)

term_id: foundation

Category: basic_concepts

Definition

인공지능에서 파운데이션 모델은 이미지, 텍스트 또는 오디오 등 광범위한 데이터를 대규모로 학습한 대형 머신러닝 모델을 지칭합니다. 이러한 모델은 다양한 다운스트림 작업에 유연하게 적용하고 미세 조정(fine-tuning)할 수 있도록 설계되었습니다.

Summary

다양한 하위 작업을 위한 기반이 되기 위해 방대한 데이터셋으로 사전 훈련된 모델을 의미합니다.

Key Concepts

  • 사전 훈련(Pre-training)
  • 파인튜닝(Fine-tuning)
  • 대규모 데이터(Large-scale data)
  • 일반화(Generalization)

Use Cases

  • 자연어 처리(Natural Language Processing)
  • 이미지 생성(Image Generation)
  • 코드 완성(Code Completion)

Related Terms

  • LLM (대형 언어 모델)
  • Transformer (트랜스포머)
  • Fine-tuning (파인튜닝)

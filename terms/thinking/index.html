Thinking

term_id: thinking

Category: basic_concepts

Definition

인간이 생물학적으로 사고하는 반면, AI의 ‘사고’는 인지 기능을 모방하는 계산적 연산을 포함합니다. 여기에는 논리적 귀납, 패턴 인식, 추론 등이 포함됩니다. 최신 대규모 언어 모델(LLM)은…

Summary

AI에서 사고란 알고리즘에 의해 시뮬레이션되는 추론, 문제 해결 및 의사 결정의 인지 과정을 의미합니다.

Key Concepts

  • 추론
  • 인지 시뮬레이션
  • 추론(Inference)
  • 문제 해결

Use Cases

  • 자연어 이해
  • 자동화된 의사 결정
  • 복잡한 문제 해결

Related Terms

  • Reasoning (추론: 주어진 정보를 바탕으로 논리적으로 결론을 도출하는 과정)
  • Cognition (인지: 지각, 기억, 추론 등을 포함한 심리적 과정)
  • Neural Networks (신경망: 인간의 뇌 구조에서 영감을 받은 계산 모델)
  • Logic (논리학: 타당한 추론의 원리와 방법을 연구하는 학문)

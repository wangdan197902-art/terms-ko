학습 곡선

term_id: learning_curve

Category: training_techniques

Definition

일반적으로 학습 곡선은 y축에 훈련 점수와 검증 점수를, x축에 훈련 샘플 수나 반복 횟수를 표시합니다. 이는 모델이 과소적합(hig…)

Summary

학습 곡선은 모델의 성능 지표를 훈련 데이터 양이나 훈련 에포크 수에 대해 그래프로 그려 학습 진행 상황을 시각화합니다.

Key Concepts

  • 훈련 점수
  • 검증 점수
  • 과소적합 vs 과대적합

Use Cases

  • 모델 성능 문제 진단
  • 샘플 크기 요구사항 결정
  • 훈련 수렴 모니터링

Related Terms

  • Validation set (검증 세트)
  • Overfitting (과대적합)
  • Convergence (수렴)

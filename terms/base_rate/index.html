기본 비율(Base Rate)

term_id: base_rate

Category: basic_concepts

Definition

통계학과 머신러닝에서 기본 비율은 주어진 데이터셋 내에서 특정 상태나 결과가 가지는 근본적인 빈도를 참조합니다. 기본 비율을 무시하면 종종 기본 비율 오류(Base rate fallacy)가 발생하여, 특정 조건 하의 사후 확률(p)

Summary

특정 증거나 테스트 결과와 무관하게, 모집단 내에서 사건이 발생할 사전 확률을 의미합니다.

Key Concepts

  • 사전 확률(Prior probability)
  • 클래스 불균형(Class imbalance)
  • 베이지안 추론(Bayesian inference)
  • 기본 비율 오류(Base rate fallacy)

Use Cases

  • 의료 진단
  • 사기 탐지
  • 위험 평가

Related Terms

  • Precision (정밀도)
  • Recall (재현율)
  • Bayes’ Theorem (베이즈 정리)

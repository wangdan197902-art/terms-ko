로봇

term_id: robot

Category: basic_concepts

Definition

로봇은 독립적으로 또는 원격 제어 하에 작업을 수행하도록 설계된 자율적 또는 반자율적인 기계 장치입니다. 일반적으로 환경 인식을 위한 센서, 작업을 수행하는 구동기(액추에이터), 그리고 이러한 요소들을 통합하여 명령을 실행하는 제어 시스템을 갖추고 있습니다.

Summary

복잡한 작업을 자동으로 수행할 수 있는 프로그래밍 가능한 기계 장치입니다.

Key Concepts

  • 자동화(Automation)
  • 구동기(Actuators)
  • 센서(Sensors)
  • 자율성(Autonomy)

Use Cases

  • 산업용 조립 라인(Industrial assembly lines)
  • 수술 보조(Surgical assistance)
  • 가정용 청소(Domestic cleaning)

Related Terms

  • 인공지능(Artificial Intelligence, 로봇의 두뇌 역할)
  • 메카트로닉스(Mechatronics, 기계공학, 전자공학, 컴퓨터 공학의 융합)
  • 자동화(Automation, 작업 자동화 기술)
  • 휴머노이드(Humanoid, 인간 형태를 띤 로봇)

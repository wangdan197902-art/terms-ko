로컬(Local)

term_id: local

Category: basic_concepts

Definition

인공지능에서 ‘로컬’은 일반적으로 원격 서버에 의존하지 않고 노트북이나 스마트폰 등 사용자의 하드웨어에서 직접 수행되는 작업을 지칭합니다. 이 접근 방식은 데이터 프라이버시를 강화하고 지연 시간을 줄입니다.

Summary

중앙 집중식 클라우드 환경이 아닌 특정 장치에서 데이터를 처리하거나 저장하는 것을 의미합니다.

Key Concepts

  • 엣지 컴퓨팅(Edge Computing)
  • 데이터 프라이버시(Data Privacy)
  • 온디바이스 처리(On-device Processing)
  • 지연 시간 감소(Latency Reduction)

Use Cases

  • 프라이버시가 중요한 음성 비서(Privacy-sensitive voice assistants)
  • 모바일 기기에서의 실시간 객체 감지(Real-time object detection on mobile devices)
  • 오프라인 AI 애플리케이션(Offline AI applications)

Related Terms

  • Cloud (클라우드: 원격 서버 기반 컴퓨팅)
  • Edge AI (엣지 AI: 장치 측에서 실행되는 인공지능)
  • Federated Learning (연관 학습: 중앙 서버 없이 분산 데이터로 학습)
  • On-premise (온프레미스: 조직 내 자체 인프라 사용)

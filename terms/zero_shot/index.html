제로 샷(Zero-shot)

term_id: zero_shot

Category: basic_concepts

Definition

제로 샷 학습은 초기 학습 단계에서 레이블이 지정된 훈련 데이터가 제공되지 않은 새로운 범주나 작업에 대해 모델이 일반화될 수 있도록 합니다. 이는 일반적으로 시맨틱 임베딩(Semantic Embedding)이나 사전 학습된 지식과 같은 외부 정보를 활용하여 달성됩니다.

Summary

미학습 클래스에 대한 작업을 수행할 수 있는 능력으로, 사전 학습 예시가 필요하지 않음.

Key Concepts

  • 일반화
  • 시맨틱 임베딩
  • 레이블 없는 데이터
  • 전이 학습

Use Cases

  • 새로운 제품 카테고리 분류
  • 미출제 주제에 대한 질문 답변
  • 크로스링구얼 텍스트 분류

Related Terms

  • few_shot (퓨어 샷: 소수의 예시를 통해 학습하는 방식)
  • transfer_learning (전이 학습: 한 도메인에서 배운 지식을 다른 도메인에 적용하는 기술)
  • generalization (일반화: 모델이 학습 데이터 외의 새로운 데이터에서도 잘 작동하는 능력)

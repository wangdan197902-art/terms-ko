스테이블 디퓨전(Stable Diffusion)

term_id: stable_diffusion

Category: application_paradigms

Definition

스테이블 디퓨전은 잠재 공간(latent space)에서 디퓨전 과정을 사용하여 텍스트 입력에 조건부로 상세한 이미지를 생성하는 딥러닝 모델입니다. 픽셀 공간 기반 디퓨전 모델과 달리 압축된 잠재 표현에서 작동하므로 계산 효율성이 높습니다.

Summary

텍스트 설명으로부터 고품질 이미지를 효율적으로 생성할 수 있는 잠재 디퓨전 모델.

Key Concepts

  • 잠재 디퓨전(Latent Diffusion)
  • 텍스트-이미지 변환(Text-to-Image)
  • 생성형 AI(Generative AI)
  • 변분 오토인코더(VAE, Variational Autoencoder)

Use Cases

  • 디지털 아트 창작
  • 개념 시각화
  • 마케팅 콘텐츠 생성

Related Terms

  • DALL-E (오픈AI의 텍스트-이미지 생성 모델)
  • Midjourney (디스코드 기반 이미지 생성 도구)
  • 디퓨전 모델(Diffusion Models) (점진적 노이즈 제거를 통한 생성 모델)
  • 생성적 적대 신경망(GANs, Generative Adversarial Networks)

OpenVINO

term_id: openvino

Category: basic_concepts

Definition

인텔이 개발한 OpenVINO(Open Visual Inference and Neural network Optimization)는 개발자가 훈련된 딥러닝 모델을 인텔 하드웨어에서 효율적으로 배포할 수 있도록 지원합니다. 이 도구에는 모델 최적화, 추론 엔진 및 다양한 하드웨어 가속 기능이 포함되어 있습니다.

Summary

OpenVINO는 인텔에서 제공하는 오픈소스 툴킷으로, 다양한 하드웨어 플랫폼에서 딥러닝 모델을 효율적으로 최적화하고 배포할 수 있게 해줍니다.

Key Concepts

  • 모델 최적화
  • 하드웨어 가속
  • 추론 엔진
  • 엣지 컴퓨팅
  • 크로스 프레임워크 지원

Use Cases

  • 인텔 CPU/GPU에서의 객체 감지 배포
  • 비디오 분석 파이프라인 최적화
  • 엣지 디바이스에서의 실시간 추론 실행

잊힐 권리

term_id: right_to_be_forgotten

Category: ethics_safety

Definition

잊힐 권리는 사용자가 자신의 개인정보가 포함된 데이터베이스 및 AI 학습 세트에서의 삭제를 요구할 수 있도록 합니다. 머신러닝에서 이를 구현하는 것은 모델이 학습 데이터의 정보를 기억하고 있을 수 있어 어려움이 따릅니다.

Summary

개인이 조직이 보유한 자신의 개인 데이터 삭제를 요청할 수 있게 하는 법적 개념입니다.

Key Concepts

  • 데이터 삭제
  • 머신 언러닝(학습 제거)
  • 법적 준수
  • 모델 재학습

Use Cases

  • GDPR(일반 데이터 보호 규정) 삭제 요청 준수
  • 모델에서 편향되거나 해로운 역사적 데이터 제거
  • AI 서비스에서 사용자 동의 철회 관리

Related Terms

  • Machine Unlearning (머신 언러닝)
  • GDPR (일반 데이터 보호 규정)
  • Data Erasure (데이터 지우기)
  • Consent Management (동의 관리)

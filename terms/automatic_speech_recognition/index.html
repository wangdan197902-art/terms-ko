자동 음성 인식

term_id: automatic_speech_recognition

Category: application_paradigms

Definition

음성 인식(ASR) 또는 음성-텍스트 변환은 인공지능을 활용하여 오디오 신호를 문자 텍스트로 변환하는 음성 처리의 하위 분야입니다. 현대 ASR 시스템은…

Summary

딥러닝 모델을 사용하여 구어체를 텍스트로 변환하는 기술입니다.

Key Concepts

  • 음향 모델링
  • 언어 모델링
  • 딥러닝
  • 전사

Use Cases

  • 음성 비서 (시리, 알렉사)
  • 영상용 실시간 자막 생성
  • 전문가를 위한 녹취 소프트웨어

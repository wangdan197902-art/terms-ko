린터(Linter)

term_id: linter

Category: engineering_practice

Definition

린터는 소스 코드를 실행하지 않고 정적 분석을 수행하는 유틸리티입니다. 문법 오류, 잠재적 버그, 코드 냄새(code smell), 그리고 스타일 가이드나 모범 사례와의 편차를 검사합니다.

Summary

프로그래밍 오류, 버그, 스타일 오류 및 의심스러운 구조를 식별하기 위해 사용되는 정적 코드 분석 도구입니다.

Key Concepts

  • 정적 분석(Static analysis)
  • 코드 품질(Code quality)
  • 스타일 강제(Style enforcement)
  • 버그 감지(Bug detection)

Use Cases

  • 자동화된 코드 리뷰(Automated code review)
  • 커밋 전 후크(Pre-commit hooks)
  • CI/CD 파이프라인 검증(CI/CD pipeline validation)

Related Terms

  • static_analysis (정적 분석)
  • code_quality (코드 품질)
  • style_guide (스타일 가이드)
  • refactoring (리팩토링)

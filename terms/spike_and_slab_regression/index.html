스파이크 앤 슬래브 회귀

term_id: spike_and_slab_regression

Category: basic_concepts

Definition

스파이크 앤 슬래브 회귀(Spike-and-slab regression)는 변수 선택과 희소 모델링을 위해 사용되는 베이지안 통계 기법입니다. 이는 두 가지 구성 요소로 이루어진 혼합 사전 분포를 사용합니다: ‘스파이크’(일반적으로 0 근처의 좁은 분포)와 ‘슬래브’(계수가 0이 아님을 나타내는 넓은 분포).

Summary

0과 0이 아닌 계수를 구분하기 위해 혼합 사전 확률 분포를 사용하는 베이지안 변수 선택 방법입니다.

Key Concepts

  • 베이지안 추론
  • 희소 모델링
  • 혼합 사전 분포
  • 변수 선택

Use Cases

  • 고차원 유전체 데이터 분석
  • 금융 위험 요인 식별
  • 예측 모델링에서의 특징 선택

Related Terms

  • Lasso (라소 회귀)
  • Ridge Regression (리지 회귀)
  • Bayesian Linear Regression (베이지안 선형 회귀)
  • Sparsity (희소성)

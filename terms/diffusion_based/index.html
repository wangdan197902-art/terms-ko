디퓨전 기반 (Diffusion-based)

term_id: diffusion_based

Category: application_paradigms

Definition

디퓨전 기반 모델은 무작위 분포에서 노이즈를 반복적으로 제거하여 새로운 데이터 샘플을 생성하는 생성형 AI의 한 클래스입니다. 이 과정은 가우시안 노이즈를 천천히 추가하는 전진 단계로 시작됩니다.

Summary

학습된 노이즈 제거 단계를 통해 점진적인 노이즈 추가 과정을 역전시켜 데이터를 생성하는 생성 모델링 접근 방식.

Key Concepts

  • 전진 과정
  • 역진 과정
  • 노이즈 제거
  • 잠재 공간

Use Cases

  • 고해상도 이미지 합성
  • 텍스트-이미지 생성
  • 의료 영상용 데이터 증강

Related Terms

  • latent_diffusion (잠재 디퓨전)

매니폴드 가설

term_id: manifold_hypothesis

Category: basic_concepts

Definition

이 가설은 차원의 저주에도 불구하고 딥러닝이 효과적으로 작동하는 이유를 설명합니다. 이미지와 같은 데이터가 백만 차원의 공간에 존재하더라도, 본질적으로 제한된 구조를 따르기 때문에…

Summary

고차원의 실제 세계 데이터는 더 높은 차원의 공간 내에서 낮은 차원의 비선형 매니폴드 위에 존재한다는 가정입니다.

Key Concepts

  • 차원 축소
  • 내재적 차원(Intrinsic Dimension)
  • 비선형 기하학
  • 일반화

Use Cases

  • 신경망 효율성 이해
  • 차원 축소 알고리즘 개발
  • 데이터 시각화 기술 개선

Related Terms

  • Principal Component Analysis (주성분 분석)
  • Autoencoders (오토인코더)
  • Curse of Dimensionality (차원의 저주)
  • Feature Learning (특징 학습)

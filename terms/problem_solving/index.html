문제 해결

term_id: problem_solving

Category: basic_concepts

Definition

인공지능에서 문제 해결은 초기 상태에서 목표 상태로 이동하기 위해 일련의 행동을 통해 체계적으로 탐색하는 접근 방식을 의미합니다. 일반적으로 문제(p)를 정의하고…

Summary

정의된 상태 공간 내에서 검색, 추론, 계획을 통해 난제에 대한 해법을 찾는 인지 과정.

Key Concepts

  • 상태 공간 탐색
  • 휴리스틱
  • 목표 상태
  • 알고리즘 설계

Use Cases

  • 로봇 및 게임 내 경로 찾기
  • 자동 정리 증명
  • 제약 만족 문제

Related Terms

  • state_space (상태 공간)
  • automated_planning (자동 계획 수립)

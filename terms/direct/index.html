직접(Direct)

term_id: direct

Category: basic_concepts

Definition

AI 맥락에서 ‘직접(Direct)‘은 강화학습의 직접 정책 최적화나 직접 매핑 등에서 볼 수 있듯이, 중간 추상화 계층을 우회하여 입력에서 출력으로 바로 연결되는 아키텍처나 추론 경로를 설명하는 데 자주 사용됩니다.

Summary

중간 복잡한 변환이나 잠재 표현 없이 입력을 출력으로 직접 매핑하는 방법이나 경로를 지칭합니다.

Key Concepts

  • 엔드투엔드(End-to-End)
  • 지연 시간 감소(Latency Reduction)
  • 단순성(Simplicity)
  • 매핑(Mapping)

Use Cases

  • 실시간 제어 시스템
  • 단순 회귀 작업
  • 엣지 AI 배포

Related Terms

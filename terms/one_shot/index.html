원샷(one-shot)

term_id: one_shot

Category: basic_concepts

Definition

원샷 학습은 알고리즘이 훈련 중에 단 하나의 긍정적 예제만 본 후 새로운 클래스나 작업에 일반화해야 하는 퓨샷 학습의 특정 유형입니다. 이 접근 방식은 인간의 학습 능력을 모방합니다.

Summary

모델이 하나의 레이블된 예제만으로 작업을 학습하는 학습 패러다임입니다.

Key Concepts

  • 퓨샷 학습(Few-shot learning)
  • 특징 추출(Feature extraction)
  • 유사도 측정(Similarity metrics)
  • 일반화(Generalization)

Use Cases

  • 제한된 샘플을 사용한 얼굴 인식
  • 필기체 식별
  • 희귀 사례에서의 의료 진단

Related Terms

  • few-shot (퓨샷)
  • zero-shot (제로샷)
  • meta-learning (메타러닝)
  • siamese networks (사이아미즈 네트워크)

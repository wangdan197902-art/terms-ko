극값 최적화

term_id: extremal_optimization

Category: training_techniques

Definition

개체군을 유지하는 유전 알고리즘과 달리 EO는 단일 솔루션에서 작동합니다. 전체 적합도에 가장 적게 기여하는 구성 요소를 식별하고 이를 무작위 대안으로 교체합니다.

Summary

극값 최적화(EO)는 자기조직화 임계 현상에서 영감을 받은 휴리스틱 검색 알고리즘으로, 성능이 가장 낮은 구성 요소를 반복적으로 제거하여 조합 최적화 문제를 해결하도록 설계되었습니다.

Key Concepts

  • 휴리스틱 검색
  • 조합 최적화
  • 자기조직화 임계 현상
  • 국소 개선

Use Cases

  • 그래프 분할 문제
  • 컴퓨팅 클러스터의 작업 스케줄링
  • 네트워크 라우팅 경로 최적화

Related Terms

  • Genetic Algorithms (유전 알고리즘)
  • Simulated Annealing (시뮬레이티드 어닐링)
  • Particle Swarm Optimization (입자 군집 최적화)
  • Metaheuristics (메타휴리스틱)

구조적(Structural)

term_id: structural

Category: basic_concepts

Definition

구조적 측면은 데이터나 신경망 레이어가 어떻게 조직되는지를 정의합니다. 그래프 신경망(GNN)에서 구조는 노드 간 연결을 의미하며, 딥러닝에서는 레이어의 위상(topology)을 의미합니다. 이해하기

Summary

시스템 내 구성 요소들의 근본적인 조직, 아키텍처 또는 배열과 관련된 개념입니다.

Key Concepts

  • 아키텍처(Architecture)
  • 위상(Topology)
  • 조직(Organization)

Use Cases

  • 그래프 신경망(Graph Neural Networks)
  • 신경망 아키텍처 탐색(Neural Architecture Search)
  • 데이터 스키마 설계(Data schema design)

Related Terms

  • Network (네트워크)
  • Layer (레이어)
  • Graph (그래프)

웹훅

term_id: webhook

Category: engineering_practice

Definition

웹훅은 특정 이벤트가 발생했을 때 한 서비스가 다른 서비스에 실시간 정보를 제공하는 메커니즘입니다. 변경 사항을 주기적으로 확인하는 폴링(polling) 방식 대신, 소스 시스템이 지정된 URL로 HTTP POST 요청을 보내어 이벤트를 알립니다.

Summary

특정 이벤트에 의해 트리거되는 사용자 정의 HTTP 콜백으로, 시스템이 다른 애플리케이션에 실시간 알림을 푸시할 수 있도록 합니다.

Key Concepts

  • 이벤트 기반
  • HTTP POST
  • 콜백 URL
  • 푸시 알림

Use Cases

  • 자동화된 CI/CD 배포
  • 결제 게이트웨이 알림
  • 슬랙 봇 통합

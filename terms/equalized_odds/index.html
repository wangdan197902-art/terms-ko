균등한 확률

term_id: equalized_odds

Category: basic_concepts

Definition

균등한 확률은 알고리즘 공정성에서 사용되는 통계적 평등 제약 조건으로, 모델이 모든 보호된 그룹에 대해 동등하게 잘 수행되도록 보장합니다. 구체적으로, 이는 특정 확률이

Summary

서로 다른 인구 통계학적 그룹 간에 참양성율(TPR)과 거짓양성율(FPR)이 동일해야 한다는 공정성 지표입니다.

Key Concepts

  • 알고리즘 공정성
  • 참양성율
  • 거짓양성율
  • 편향 완화

Use Cases

  • 공정한 대출 알고리즘
  • 채용 추천 시스템
  • 형사 사법 리스크 평가

Related Terms

  • Demographic Parity (인구통계학적 평등)
  • Individual Fairness (개별 공정성)
  • Bias Audit (편향 감사)
  • Fairness Constraints (공정성 제약)

A/B 테스트

term_id: ab_testing

Category: engineering_practice

Definition

A/B 테스트는 두 가지 변형(A와 B)을 무작위 통제 실험으로 비교하여 특정 지표에서 더 나은 결과를 가져오는지를 평가합니다. AI 엔지니어링에서 이는 최적화에

Summary

두 가지 변수 버전을 비교하여 더 나은 성능을 결정하는 통계적 방법입니다.

Key Concepts

  • 통제 그룹
  • 통계적 유의성
  • 가설 검정
  • 무작위화

Use Cases

  • 추천 엔진 클릭률 최적화
  • 정확도를 위한 서로 다른 모델 아키텍처 비교
  • AI 기반 애플리케이션의 UI 변경 테스트

Related Terms

  • Multivariate Testing (다변량 테스트)
  • Statistical Power (통계적 힘)
  • Conversion Rate Optimization (전환율 최적화)
  • Randomized Controlled Trial (무작위 통제 시험)

지각(Percept)

term_id: percept

Category: basic_concepts

Definition

지각은 외부 자극이 지각 시스템을 거쳐 처리된 후 내부적으로 형성된 표현입니다. 인공지능에서 이는 저수준의 신호 처리 과정을 거친 후 생성된 구조화된 데이터 출력을 해당하며, 시스템이 환경에 대해 갖는 인식의 단위라고 볼 수 있습니다.

Summary

인지 모듈에 의해 처리된 감각 입력의 정신적 표현 또는 해석된 결과.

Key Concepts

  • 표현(Representation)
  • 해석(Interpretation)
  • 의미론적 의미(Semantic meaning)
  • 데이터 추상화(Data abstraction)

Use Cases

  • 객체 감지 출력 결과
  • 음성 인식 텍스트 변환
  • 의료 영상 진단 보조 도구

Related Terms

  • Feature vector (특성 벡터)
  • Symbol grounding (기호 고정)
  • Cognitive map (인지 지도)

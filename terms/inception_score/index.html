인셉션 스코어 (Inception Score)

term_id: inception_score

Category: basic_concepts

Definition

인셉션 스코어(IS)은 생성 적대 신경망(GAN) 및 기타 생성 모델의 성능을 평가하기 위해 도입된 통계적 측정치입니다. 이는 두 가지 요소를 결합합니다: 이미지 품질(선명도)과…

Summary

생성된 이미지의 선명도와 다양성을 측정하여 이미지 품질을 평가하는 지표입니다.

Key Concepts

  • 생성 모델 (Generative Models)
  • 이미지 품질 (Image Quality)
  • 다양성 지표 (Diversity Metric)
  • GAN 평가 (GAN Evaluation)

Use Cases

  • GAN 성능 평가
  • 생성 모델 아키텍처 비교
  • 이미지 합성 품질 벤치마킹

Related Terms

  • Fréchet Inception Distance (FID)
  • Precision and Recall (정밀도와 재현율)
  • Generative Adversarial Networks (GANs)
  • Image Synthesis (이미지 합성)

퓨 샷 러닝

term_id: few_shot_learning

Category: training_techniques

Definition

퓨 샷 러닝은 인간 학습의 효율성을 모방하여, 단 몇 개의 예시만으로 모델을 일반화할 수 있도록 하는 것을 목표로 합니다. 이는 일반적으로 메타러닝(Meta-learning) 전략에 의존하며, 모델이 다양한 하위 태스크를 사전에 경험하거나 학습하여 새로운 태스크에 빠르게 적응하도록 설계됩니다.

Summary

퓨 샷 러닝은 모델이 매우 제한된 라벨링된 학습 데이터로부터 새로운 개념을 학습하는 머신러닝 패러다임입니다.

Key Concepts

  • 메타러닝
  • 데이터 효율성
  • 일반화
  • 태스크 적응

Use Cases

  • 희귀 질병 식별
  • 새로운 제품 분류
  • 맞춤형 객체 감지

Related Terms

  • meta_learning (메타러닝: 학습 방법을 학습하는 기술)
  • transfer_learning (전이 학습: 한 영역에서 배운 지식을 다른 영역에 적용)
  • data_efficiency (데이터 효율성: 적은 데이터로 높은 성능 달성)
  • supervised_learning (지도 학습: 라벨이 있는 데이터로 학습)

인용

term_id: citation

Category: application_paradigms

Definition

생성형 AI 모델이 콘텐츠를 생산함에 따라 학술적 무결성과 법적 준수를 유지하기 위해 인용 메커니즘의 필요성이 대두되었습니다. 이는 생성된 콘텐츠 내에 원래 출처에 대한 참조를 포함시키는 것을

Summary

AI에서의 인용은 투명성과 지적 재산권 준수를 보장하기 위해 생성된 텍스트나 모델 내에서 사용된 원자료나 데이터를 출처에 귀속시키는 관행을 의미합니다.

Key Concepts

  • 귀속
  • 환각(Hallucination) 완화
  • 저작권 준수
  • 출처 검증

Use Cases

  • 학술 연구 보조 도구
  • 법률 문서 생성
  • 팩트체크 도구

Related Terms

  • RAG (검색 증강 생성)
  • Grounding (그라운딩/근거 부여)
  • Knowledge Graph (지식 그래프)
  • Intellectual Property (지적 재산권)

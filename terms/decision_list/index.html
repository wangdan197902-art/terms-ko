결정 목록(Decision list)

term_id: decision_list

Category: basic_concepts

Definition

결정 목록은 지식을 조건부 규칙의 시퀀스로 표현하는 머신러닝 모델 유형입니다. 각 규칙은 조건과 예측된 클래스 레이블로 구성됩니다. 새로운 데이터를 분류할 때, 입력과 일치하는 첫 번째 규칙이 해당 데이터의 클래스를 결정합니다.

Summary

순서가 지정된 규칙 목록으로 구성된 해석 가능한 분류기로, 첫 번째로 일치하는 규칙이 출력을 결정합니다.

Key Concepts

  • 순서 있는 규칙
  • 해석 가능성
  • 일치 우선 세맨틱스
  • 규칙 기반 분류

Use Cases

  • 의료 진단 지원
  • 신용 점수 설명
  • 법적 준수 검사

Related Terms

  • Decision tree (결정 트리)
  • Rule-based system (규칙 기반 시스템)
  • If-then rules (if-then 규칙)
  • Interpretable ML (해석 가능한 머신러닝)

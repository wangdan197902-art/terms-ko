마스크 생성

term_id: mask_generation

Category: application_paradigms

Definition

마스크 생성은 특정 작업 중 데이터 세트의 어떤 요소가 표시되거나 활성화될지를 결정하는 공간적 또는 시간적 마스크를 생성하는 것을 포함합니다. 컴퓨터 비전에서는 객체 분할 및…

Summary

모델 처리 과정에서 입력 데이터의 일부를 선택적으로 숨기거나 강조하기 위해 이진 또는 확률적 마스크를 생성하는 과정입니다.

Key Concepts

  • 이진 마스킹(Binary masking)
  • 어텐션 마스크(Attention masks)
  • 인페인팅(Inpainting)
  • 특징 선택(Feature selection)

Use Cases

  • 이미지 인페인팅 및 복원
  • 트랜스포머 어텐션 메커니즘
  • 객체 감지 및 분할

Related Terms

  • Attention mechanism (어텐션 메커니즘)
  • Semantic segmentation (의미론적 분할)
  • Inpainting (인페인팅)
  • Feature masking (특징 마스킹)

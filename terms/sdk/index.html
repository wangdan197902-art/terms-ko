SDK(소프트웨어 개발 키트)

term_id: sdk

Category: engineering_practice

Definition

SDK는 특정 플랫폼이나 서비스를 위한 애플리케이션을 개발할 수 있도록 하는 소프트웨어 개발 도구 모음입니다. AI 분야에서 SDK는 사전 구축된 라이브러리, API 및 유틸리티를 제공하여…

Summary

애플리케이션 구축을 위한 도구, 라이브러리 및 문서를 제공하는 소프트웨어 개발 키트입니다.

Key Concepts

  • API 통합
  • 라이브러리
  • 개발 도구
  • 추상화

Use Cases

  • 앱에 대규모 언어 모델(LLM) 통합
  • 컴퓨터 비전 파이프라인 구축
  • 프로덕션 환경에 머신러닝 모델 배포

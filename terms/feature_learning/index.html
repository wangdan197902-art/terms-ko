피처 학습 (Feature Learning)

term_id: feature_learning

Category: training_techniques

Definition

딥러닝과 밀접하게 연관된 피처 학습은 수동 피처 엔지니어링에 의존하는 대신, 모델이 계층적 표현을 직접 학습하도록 합니다. 여러 층(layer)을 통해 원시 입력 데이터에서 복잡한 패턴과 추상화를 자동으로 추출하여 더 나은 성능을 달성합니다.

Summary

알고리즘이 원시 데이터로부터 감지 또는 분류에 필요한 피처를 자동으로 발견하는 접근 방식입니다.

Key Concepts

  • 자동 표현
  • 계층적 층
  • 딥러닝
  • 수동 엔지니어링 배제

Use Cases

  • 이미지 인식 및 객체 탐지
  • 음성-텍스트 변환
  • 자연어 이해 작업

Related Terms

  • Deep Learning (딥러닝)
  • Feature engineering (피처 엔지니어링)
  • Autoencoder (오토인코더)
  • Embedding (임베딩)

브래들리-테리 모델

term_id: bradleyterry_model

Category: basic_concepts

Definition

브래들리-테리 모델은 심리측정학과 머신러닝에서 쌍별 비교를 처리하기 위해 널리 사용되는 확률 모델입니다. 이 모델은 각 항목에 잠재 점수를 할당하여, 특정 항목이 다른 항목보다 선호될 확률을 계산합니다. 일반적으로 로지스틱 회귀의 일종으로 간주되며, 순위 데이터나 선호도 데이터를 모델링하는 데 효과적입니다.

Summary

쌍별 비교 데이터를 분석하여 한 항목이 다른 항목보다 선호될 확률을 추정하는 데 사용되는 통계 모델입니다.

Key Concepts

  • 쌍별 비교
  • 순위 시스템
  • 로지스틱 회귀
  • 선호도 학습

Use Cases

  • 스포츠 순위 알고리즘
  • A/B 테스트 분석
  • 인간 피드백을 통한 강화 학습(RLHF)

Related Terms

  • Elo Rating (엘로 레이팅:棋士나 선수의 실력을 수치화한 평가 시스템)
  • Plackett-Luce Model (플래킷-루스 모델: 순서 있는 선택 데이터를 다루는 확률 모델)
  • Isotonic Regression (등온 회귀: 단조성을 제약 조건으로 하는 회귀 분석)
  • RLHF (인간 피드백을 통한 강화 학습)
